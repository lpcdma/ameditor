# ameditor
copy from https://github.com/ele7enxxh/AmBinaryEditor, fix bug,change build.

# Usage

*nix
```
mkdir build
cd build
cmake ..
make
```

Win32

```
git clone https://github.com/lpcdma/ameditor.git
cd ameditor
mkdir build
cd build
cmake.exe .. -G "Visual Studio 14 2015 Win64"
```
open ameditor.sln
